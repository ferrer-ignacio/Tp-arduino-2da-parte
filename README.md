#Trabajo Práctico: Periféricos Parte 2 - Informática II
## Descripción
El sistema permite la comunicación bidireccional por puerto serie:
1. Arduino a PC: Envía el estado de los pulsadores físicos para visualizarlos en la pantalla.
2. PC a Arduino: Permite encender/apagar LEDs haciendo clic en la interfaz gráfica.

Esquema de Conexión (Hardware)
Se utilizó la siguiente configuración de pines en la placa Arduino:

Entradas (Pulsadores):
    * Pulsador 1: Pin D2
    * Pulsador 2: Pin D3
Salidas (LEDs):
    * LED 1: Pin D12
    * LED 2: Pin D13
